# 项目简介

## BuyYouWant 是什么

BuyYouWant 是一个基于 **Spring Cloud 微服务架构** 的全功能电商平台，对标淘宝核心业务，涵盖从商品浏览、购物车、下单、支付到物流跟踪的完整购物链路。项目分为 **用户端商城**（Nuxt.js SSR）和 **管理后台**（Vue3 + Element Plus）两套前端，后端拆分为 12 个微服务，通过 OpenFeign 同步调用与 RocketMQ 异步事件驱动实现服务间协作，采用 Seata AT 模式保障分布式事务一致性，是一套可学习、可运行、可扩展的企业级微服务电商解决方案。

## 核心功能

### 用户中心
- 手机号/密码注册、登录，JWT Token 鉴权
- 个人资料管理、头像上传
- 收货地址 CRUD、默认地址设置
- 会员等级体系与积分管理

### 商品中心
- 三级分类管理（一级/二级/三级）
- 品牌管理与关联分类
- SPU / SKU 多规格体系（规格属性、价格、库存独立管理）
- Elasticsearch 全文搜索（IK 中文分词、多条件聚合筛选）
- 商品库存管理与预扣

### 购物车
- Redis Hash + MySQL 双写保障
- 加购、修改数量、删除商品
- 全选/单选、批量结算
- 实时价格计算与库存校验

### 订单中心
- 雪花算法生成全局唯一订单号
- Seata 分布式事务保障下单链路一致性
- 完整订单状态机（待付款 → 待发货 → 待收货 → 已完成/已取消）
- 延迟队列实现超时自动取消（30 分钟未支付）

### 支付中心
- 策略模式支持多种支付方式（支付宝 / 微信）
- 模拟支付回调流程
- RocketMQ 异步通知订单服务更新状态
- 支付流水记录与对账

### 物流中心
- 发货管理（关联快递公司、填写运单号）
- 物流轨迹跟踪（模拟）
- 物流状态更新与 RocketMQ 通知

### 评价系统
- MySQL 存储评价基础数据 + MongoDB 存储评价详情与图片
- 多维度评分（描述相符、物流速度、服务态度）
- 图片评价上传
- 评分统计与聚合展示

### 营销中心
- 优惠券：满减券、折扣券、无门槛券，领取/使用/过期管理
- 秒杀活动：Redis Lua 脚本原子预扣库存 + Sentinel 限流保护 + RocketMQ 异步下单
- 拼团活动：开团/参团/成团逻辑

### 管理后台
- Dashboard 数据看板（订单量、销售额、用户数等 ECharts 图表）
- 商品管理、分类管理、品牌管理
- 订单管理、发货管理
- 优惠券管理、秒杀活动管理
- 用户管理、评价管理

## 系统特点

- **微服务架构**：12 个独立部署的微服务，职责清晰，通过 Nacos 实现服务注册发现与配置管理
- **分布式事务**：Seata AT 模式保障下单核心链路（扣库存 + 核销优惠券 + 清购物车 + 创建订单）的数据一致性
- **高性能**：Redis Lua 脚本实现库存原子预扣，避免超卖；RocketMQ 异步削峰落库，提升吞吐量
- **全文搜索**：Elasticsearch + IK 中文分词，支持商品名称、描述、属性的多维度搜索与聚合筛选
- **消息驱动**：RocketMQ 事件驱动实现跨服务最终一致性（支付通知、库存扣减、物流状态变更等）
- **安全防护**：JWT + Gateway 全局过滤器鉴权，Sentinel 限流熔断保护核心接口
- **API 文档**：各微服务集成 Knife4j 在线调试文档，Gateway 通过 SpringDoc 聚合所有服务的 API 文档
